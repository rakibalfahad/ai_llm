## Quick Start

### Build

```bash
bash create_docker.sh
```

Generates the Dockerfile, builds `deeplearning:v100-llm`, and runs a GPU verification test.

### Run

```bash
# Interactive shell
docker run --rm --gpus all \
  -v $(pwd)/data:/workspace/data \
  -it deeplearning:v100-llm

# Jupyter Lab
docker run --rm --gpus all \
  -p 8888:8888 \
  -v $(pwd)/data:/workspace/data \
  deeplearning:v100-llm \
  jupyter lab --ip=0.0.0.0 --allow-root --no-browser

# Gradio app
docker run --rm --gpus all \
  -p 7860:7860 \
  -v $(pwd)/data:/workspace/data \
  deeplearning:v100-llm \
  python3 your_app.py
```

## Example: Train MNIST on GPU

Two example scripts are included in `scripts/` — one for PyTorch, one for TensorFlow.

### Run with volume mounts

Mount `scripts/` and `data/` into the container so your code and outputs persist on the host:

  -v $(pwd)/data:/workspace/data \
  deeplearning:v100-llm \
  python3 /workspace/scripts/pt_mnist_gpu.py

## Common Workflows

### Load a quantized LLM (fits in V100 16 GB)

```python
from transformers import AutoModelForCausalLM, AutoTokenizer, BitsAndBytesConfig

quantization_config = BitsAndBytesConfig(load_in_4bit=True)

model = AutoModelForCausalLM.from_pretrained(
    "meta-llama/Llama-2-7b-chat-hf",
    quantization_config=quantization_config,
    device_map="auto",
)
tokenizer = AutoTokenizer.from_pretrained("meta-llama/Llama-2-7b-chat-hf")
```

### Fine-tune with LoRA

```python
from peft import LoraConfig, get_peft_model

lora_config = LoraConfig(r=16, lora_alpha=32, target_modules=["q_proj", "v_proj"])
model = get_peft_model(model, lora_config)
model.print_trainable_parameters()
```

### Quick PyTorch GPU check

```python
import torch
print(torch.cuda.is_available())        # True
print(torch.cuda.get_device_name(0))    # Tesla V100-PCIE-16GB
```

## Troubleshooting

### CUDA not available in PyTorch

```bash
# Verify nvidia runtime is default
docker info | grep "Default Runtime"   # should show: nvidia

# Reconfigure if needed
sudo nvidia-ctk runtime configure --runtime=docker --set-as-default
sudo systemctl restart docker
```

### Docker Hub rate limit (429 Too Many Requests)

The image pulls from `nvcr.io` (NVIDIA registry), not Docker Hub. If you see rate limits, ensure the Dockerfile `FROM` line uses `nvcr.io/nvidia/cuda:`.

### Out of GPU memory

V100 has 16 GB. Use 4-bit quantization (`load_in_4bit=True`) for models > 3B parameters.

## Save & Reuse the Image

### Export to a tar file

```bash
docker save deeplearning:v100-llm | gzip > deeplearning-v100-llm.tar.gz
```

### Load on another machine (or after a Docker reset)

```bash
docker load < deeplearning-v100-llm.tar.gz
```

### Commit a running container (preserve installed packages / downloaded models)

```bash
# Find the container ID
docker ps

# Save its current state as a new image
docker commit <container_id> deeplearning:v100-llm-custom
```

### Tag & push to a private registry

```bash
docker tag deeplearning:v100-llm registry.example.com/deeplearning:v100-llm
docker push registry.example.com/deeplearning:v100-llm

# Pull on another host
docker pull registry.example.com/deeplearning:v100-llm
```
